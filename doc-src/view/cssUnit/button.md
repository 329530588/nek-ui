### 示例
#### 基本形式

在`<button>`或`<a>`标签上使用`class="u-btn"`，两者样式完全一致。

<div id="j-example1">
<button class="u-btn">Button</button>
<a class="u-btn">aButton</a>
<a class="u-btn" href="#">aButton#</a>
</div>

```html
<button class="u-btn">Button</button>
<a class="u-btn">aButton</a>
<a class="u-btn" href="#">aButton#</a>
```
