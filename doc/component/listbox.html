列表框 Listbox

示例

基本形式

<listbox source={source} />

var source = [
    {id: 0, name: '第一项'},
    {id: 1, name: '第二项'},
    {id: 2, name: '第三项'}
];

var listbox = new Listbox({
    data: {
        source: source
    }
}).$inject('#j-example1');

禁用

<listbox source={source} disabled={true} />

var source = [
    {id: 0, name: '第一项'},
    {id: 1, name: '第二项'},
    {id: 2, name: '第三项'}
];

var listbox = new Listbox({
    data: {
        source: source,
        disabled: true
    }
}).$inject('#j-example1');

API

Class

Listbox

继承自Component。

Options

new Listbox({options})
参数类型默认值描述
dataobject绑定属性
data.sourceobject[][]数据源
data.source[].idnumber每项的id
data.source[].namestring每项的内容
data.selectedobjectnull选择项
data.disabledbooleanfalse是否禁用该组件

Methods

listbox.select(item)

选择某一项

参数类型默认值描述
itemobject选择项

Events

listbox.$on('select', function($event))

选择某一项时触发

属性类型描述
selectedobject选中项
