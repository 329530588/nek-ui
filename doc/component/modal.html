模态对话框 Modal

含有遮罩层的对话框，用于模拟浏览器的alert、confirm和prompt。

模态对话框通过遮罩层来阻止用户的其他行为。

示例

基本形式

var modal = new Modal({
    data: {
        title: 'Modal标题',
        content: 'Modal内容'
    }
});

Alert

Modal.alert('Alert内容');

Confirm

Modal.confirm('Confirm内容');

API

Class

Modal

继承自Component。

Options

new Modal({options})
参数类型默认值描述
dataobject绑定属性
data.titlestring'提示'对话框标题
data.contentstring''对话框内容
data.okButtonstring | booleantrue确定按钮的文字。值为字符串时显示文字为该字符串，值为`true`时显示文字为“确定”，值为否定时不显示确定按钮。
data.cancelButtonstring | booleanfalse取消按钮的文字。值为字符串时显示文字为该字符串，值为`true`时显示文字为“取消”，值为否定时不显示确定按钮。
data.widthnumbernull对话框宽度。值为否定时宽度为CSS设置的宽度。
okfunction当点击确定的时候执行
cancelfunction当点击取消的时候执行

Methods

modal.close(result)

关闭模态对话框

参数类型默认值描述
resultboolean点击确定还是取消

Static Methods

Modal.alert([content][, title])

弹出一个alert对话框。关闭时始终触发确定事件。

参数类型默认值描述
contentstring''对话框内容
titlestring'提示'对话框标题
Modal.confirm([content][, title])

弹出一个confirm对话框

参数类型默认值描述
contentstring''对话框内容
titlestring'提示'对话框标题

Events

modal.$on('close', function($event))

关闭对话框时触发

属性类型描述
resultboolean点击了确定还是取消
modal.$on('ok', function($event))

确定对话框时触发

属性类型描述
modal.$on('close', function($event))

取消对话框时触发

属性类型描述
