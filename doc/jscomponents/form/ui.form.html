示例

基本形式

<ui.form service="{service.selector}" ref="formgroup">
    <form.item title="标题1" cols=12 sourceKey="warehouse">
        <ui.select />
    </form.item>
    <form.item title="标题2" cols=12>
        <ui.input />
    </form.item>
</ui.form>

var component = new RGUI.Component({
    template: template,
    data: {
        service: {
            selector: 'api'
        }
    }
});

API

Class

UIForm

继承自Validation。

Options

new UIForm()
参数类型默认值数据流向描述
dataobject绑定属性
data.servicestring''outer => inner全站异步获取source的接口地址
data.sourcePathstring'data'outer => inner获取到select数据后,读取json数据的路径

Inherited Methods

uIForm.validate()

验证所有表单组件

参数类型默认值描述
无参数
返回值类型描述
conclusionobject结论
