API

Class

TreeView

继承自SourceComponent。

Options

new TreeView({options})
参数类型默认值描述
dataobject绑定属性
data.sourceobject[][]数据源
data.source[].idnumber每项的id
data.source[].namestring每项的内容
data.selectedobjectnull当前选择项
data.hierarchicalbooleanfalse是否分级动态加载，需要service
data.readonlybooleanfalse是否只读
data.disabledbooleanfalse是否禁用
data.visiblebooleantrue是否显示
data.classstring''补充class
serviceobject数据服务

Methods

treeView.select(item)

选择某一项

参数类型默认值描述
itemobject选择项

Events

on-select={$event}

选择某一项时触发

属性类型描述
selectedobject当前选择项
on-toggle={$event}

展开或收起某一项时触发

属性类型描述
itemobject展开收起项
openboolean展开还是收起
