基本形式

API

Class

InputEx

继承自Component。

Options

new InputEx({options})
参数类型默认值描述
dataobject绑定属性
data.sourceobject[][]数据源
data.source[].idnumber每项的id
data.source[].namestring每项的内容
data.selectedobjectnull当前选择项
data.disabledbooleanfalse是否禁用该组件

Methods

inputEx.select(item)

选择某一项

参数类型默认值描述
itemobject选择项

Events

on-select={$event}

选择某一项时触发

属性类型描述
selectedobject当前选择项
