API

Class

Menu

继承自Dropdown。

Options

new Menu()
参数类型默认值描述
dataobject绑定属性
data.sourceobject[][]数据源
data.source[].idnumber每项的id
data.source[].namestring每项的内容
data.source[].disabledbooleanfalse禁用此项
data.source[].dividerbooleanfalse设置此项分隔线
data.itemTemplatestringnull单项模板
data.openbooleanfalse当前为展开/收起状态
data.disabledbooleanfalse是否禁用
data.visiblebooleantrue是否显示
data.classstring''补充class
serviceobject数据服务

Inherited Methods

menu.select(item)

选择某一项

参数类型默认值描述
itemobject选择项
menu.toggle(open)

在展开/收起状态之间切换

参数类型默认值描述
openboolean展开/收起
menu.$updateSource()

从service中更新数据源

参数类型默认值描述
无参数

Events

menu.$on('toggle')

展开或收起某一项时触发

属性类型描述
itemobject展开收起项
openboolean展开还是收起

Inherited Events

menu.$on('select')

选择某一项时触发

属性类型描述
selectedobject当前选择项
