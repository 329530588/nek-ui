表格视图 TableView

示例

基本形式

<tableView source={source} fields={@(fields)} />

或

var source = [
    {id: 3, name: '精通Javascript开发', org: '前端Funs', hits: 42371},
    {id: 2, name: 'Android深入浅出', org: 'Android学院', hits: 30645},
    {id: 1, name: 'cocos2dx游戏开发教程', org: '鱼C课程', hits: 25112},
    {id: 4, name: 'MySQL数据库', org: 'LAMP兄弟连', hits: 18089},
    {id: 5, name: 'Arduino初级教程', org: '硬件社', hits: 16361}
];

var tableView = new TableView({
    data: {
        source: source,
        fields: [
            {key: 'name', name: '课程'},
            {key: 'org', name: '机构'},
            {key: 'hits', name: '点击率'}
        ]
    }
}).$inject('#j-example1');

远程数据

同ListBox。

排序

只要给可排序的字段添加sortable: true即可。

如果有service属性，会向服务器发送请求并带上排序相关的参数，进行后端排序；如果没有service属性，则在本地对source进行排序。

<tableView source={source} fields={@(fields)} />

或

var source = [
    {id: 3, name: '精通Javascript开发', org: '前端Funs', hits: 42371},
    {id: 2, name: 'Android深入浅出', org: 'Android学院', hits: 30645},
    {id: 1, name: 'cocos2dx游戏开发教程', org: '鱼C课程', hits: 25112},
    {id: 4, name: 'MySQL数据库', org: 'LAMP兄弟连', hits: 18089},
    {id: 5, name: 'Arduino初级教程', org: '硬件社', hits: 16361}
];

var tableView = new TableView({
    data: {
        source: source,
        fields: [
            {key: 'name', name: '课程', sortable: true},
            {key: 'org', name: '机构'},
            {key: 'hits', name: '点击率', sortable: true}
        ]
    }
}).$inject('#j-example2');

API

Class

TableView

继承自SourceComponent。

Options

new TableView({options})
参数类型默认值描述
dataobject绑定属性
data.sourceobject[][]数据源
data.source[].idnumber每项的id
data.source[].namestring每项的内容
data.fieldobject[][]字段集
data.field[].keystring每个字段的key
data.field[].namestring每个字段在表头显示的文字，如果没有则显示key
data.stripedbooleanfalse是否显示条纹
data.hoverbooleanfalse是否每行在hover时显示样式
data.classstring''补充class

Methods

tableView.sort(field)

按照某个字段排序

参数类型默认值描述
fieldobject排序字段

Events

on-sort={$event}

按照某个字段排序时触发

属性类型描述
fieldobject排序字段
