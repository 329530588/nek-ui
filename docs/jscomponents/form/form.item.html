示例

基本形式

<div class="f-cb">
    <form.item cols=4 offset=1>
        <ui.input placeholder="请输入" />
    </form.item>
</div>

var component = new RGUI.Component({
    template: template,
    data: {}
});

表单项

在表单中使用

<ui.form>
    <form.item cols=12 title="用户名" hint="用户名的用途">
        <ui.input placeholder="请输入" />
    </form.item>
</ui.form>

var component = new RGUI.Component({
    template: template,
    data: {
        source: [
            {name: '选项1'},
            {name: '选项2'},
            {name: '选项3'},
            {name: '选项4'},
            {name: '选项5'},
            {name: '选项6'}
        ]
    }
});

表单项

在表单中使用时label横向放置

<ui.form>
    <form.item cols=12 title="用户名" hint="用户名的用途"  transverse>
        <ui.input placeholder="请输入" />
    </form.item>
</ui.form>

var component = new RGUI.Component({
    template: template,
    data: {
        source: [
            {name: '选项1'},
            {name: '选项2'},
            {name: '选项3'},
            {name: '选项4'},
            {name: '选项5'},
            {name: '选项6'}
        ]
    }
});

API

Class

FormItem

继承自Validation。

Options

new FormItem()
参数类型默认值数据流向描述
dataobject绑定属性
data.titlestring''outer => innerlabel显示的文字
data.colsnumberouter => inner布局列数
data.labelColsnumber4outer => inner如果有title, label占的列数
data.offsetnumberouter => inner布局offset
data.rowstring''outer => inner垂直布局row
data.requiredbooleanfalseouter => inner是否必选项
data.tipstring''outer => inner字段说明

Inherited Methods

formItem.validate()

验证所有表单组件

参数类型默认值描述
无参数
返回值类型描述
conclusionobject结论
