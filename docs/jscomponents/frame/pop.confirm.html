基本形式

    template: template

事件(打开console, 查看输出)

<pop.confirm content="Are you sure delete this task?" on-ok={this.onok()}>
    <button class="u-btn">保存提交</button>
</pop.confirm>

var component = new RGUI.Component({
    template: template,
    onok: function() {
        console.log(123);
    }
});

自定义模板(打开console, 查看输出)

<pop.confirm contentTemplate={testTemplate} on-ok={this.onok($event)}>
    <button class="u-btn">保存提交</button>
</pop.confirm>

var component = new RGUI.Component({
    template: template,
    config: function() {
        this.data.testTemplate = '<form.item ref="validation" title="备注" required row><ui.textarea required showTip=false value={remark} height=50 /></form.item>';
    },
    onok: function(json) {
        console.log(json.data.remark);
        console.log(json.sender);
    }
});

API

Class

PopConfirm

继承自Component。

Options

new PopConfirm()
参数类型默认值数据流向描述
dataobject绑定属性
data.contentstring''outer => inner弹窗中的文本内容
data.contentTemplatestring''outer => inner弹窗中的模板内容,回调中会将PopConfirm的data返回;
data.placementstring'top'outer => innertips展示出的位置：top left right bottom topLeft topRight bottomLeft bottomRight leftTop leftBottom rightTop rightBottom
data.okTextstring'确定'outer => innerok按钮文案
data.cancelTextstring'取消'outer => inner取消按钮文案
data.hideWhenScrollboolean'false'outer => innerwindow滚动时,是否影藏popover

Events

popConfirm.$on('ok')

确定时触发

属性类型描述
senderobject事件发送对象
dataobjectpopConfirm组件的数据
popConfirm.$on('cancel')

取消时触发

属性类型描述
senderobject事件发送对象
dataobjectpopConfirm组件的数据
