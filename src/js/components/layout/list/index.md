## 基本形式

对`ul`或`ol`标签添加`.m-list`，将不显示默认的列表样式。

<!-- demo_start -->
<div class="m-example"></div>

```html
<ul class="m-list">
    <li>List Item</li>
    <li>List Item</li>
    <li>List Item</li>
    <li>List Item</li>
    <li>List Item</li>
</ul>
```
<!-- demo_end -->
